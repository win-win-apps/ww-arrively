## Convention reminder

This file is a **scratchpad** — not a backlog and not a PR description. Update at the end of each working session with:
1. **Last Session** — date + what landed and current state
2. Anything *unfinished* or *follow-up* worth surfacing next session
3. Bugs/observations not yet ready for formal tickets
4. Next-steps queue (`- [ ]` checklist)

Cross-team tracking goes in GitHub issues or the task system. "What the next session needs to know" goes here.

When the same bug/observation graduates to a real ticket, remove it from TODO.md and reference the ticket. Keep this file under ~150 lines.

**Read TODO.md first when starting a new session.**
